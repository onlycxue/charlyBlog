写在前面

这里主要介绍一下php对文件的处理 、php的函数、以及php的面向对象。学习过程主要是先看书，然后用代码验证，最后
到现在的博客整理，写博客的目的主要为了方便日后的复习，和希望对像我一样菜鸟级别的程序猿有所帮助。

文件的操作

对文件的操作过程大致都抽象3个步骤 1) 打开文件 , 2) 执行操作(读/写) , 3) 关闭文件。针对这几个操作php都做
比较好用的api。

  1. fopen()打开文件。

    php打开的文件可以是相对路径或绝对路径，也可以是网络文件(http,http和ftp)。打开方式和c语言的类似。

  2. fclose()关闭文件

  3. fwrite()写入函数

  4. fread()读取函数

  5. fgets()按行读取函数

    fgets()函数是按行读取，优先查找换行符“\n”，也就是说如果定义了读取字节的个数num,如果’\n’
    的位置在num之前会返回’\n’之前的内容，否则返回num个字符

  6. fgetss()与 fgets()的不同之处是可以过滤字符串中php和html的标记符

  7. fgetcsv()用来 读取csv格式的文件并转为array

  8. 还有几个便捷的函数一定要说，直接上代码

函数的操作

以下将从函数的命名，传参，和函数内变量的作用介绍一下php的函数与其他语言(c++) 的不同之处。

  1. php函数的名字是不区分大小写的

  2. 不限参数传值

  3. php按值传递和c++类似,下面看一下按引用传值

  4. php的默认参数

    注意当使用默认参数时，任何默认参数必须放在任何非默认参数的右侧；否则，函数将不会按照预期的情况工作。

  5. 用变量存储函数名

面向对象

谈到面向对象都离不开它的三大特征,封装、继承、多态,学过c++ 或 java的童鞋学这部分就会轻车熟路。

  1. 一个简单类

  2. php类的访问修饰符与c++ 类似有 public,private,protected 三种。

  3. php类中常量的定义与访问

  4. php内置方法 __construct、__destruct 顾名思义,构造函数和析构函数 __clone会在执行
    克隆对象时调用

  5. php的继承更接近java,也有抽象类，和接口的概念，继承抽象类使用extends, 继承接口使用implements。

    抽象类和普通类的不同是，抽象类 类名前要用 abstract 修饰，类中至少有一个抽象函数(即用abstract修饰的函数)

结束语

变量和赋值

php有8种基本类型。
四种标量类型：boolean(布尔类型)、integer(整型)、float(浮点型)、string(字符串)
两种复合类型：array(数组)、object(对象)
两种特殊的类型: resource(资源)、NULL
然而在实际使用php我们不用像c++ ,java 那样静态的去声明变量，php会根据赋的值自动推导类型

  2. 引用赋值

    这里的引用赋值和c++类似，相当于一个变量的别名。

  3. 可变变量

    单引号和双引号在php中处理的结果是不同的,双引号的内容可以被解释而且替换而单引号的内容会认为是普通字符

  4. 定义常量

    php与c/c++类似也有一个const关键字,稍后我在做一下区分

  5. magic变量
    php内置了一些方便我们使用的变量，如: __FILE__、__METHOD__、__FUNCTION__等等

运算符

  2. 比较运算符

    不等于的比较运算符(!= !==)和上面相似， == 属于弱比较(内容相同就为true,不考虑变量类型)而 === 属于强比较(不仅要求内容相同,而且类型也要相同才能为true).

数组 array

array类型在php中是最常用到的，它的特点和lua的table很类似，php中的数组实际上是一个有序映射。映射是一种把values 关联到keys 的类型。此类型在很多方面做了优化，因此可以把它当成真正的数组，或列表(向量), 散列表(是映射的一种实现)，字典，集合，栈，队列，以及更多可能性。

数组元素的清除

  1. array的 key 可以是 integer 或者 string。 value 可以是任意类型。

    是不是有点感觉很惊讶，让人琢磨不透，不要着急弄清php的array的覆盖规则就豁然开亮了。
    a.包含有合法整形值的字符串会被转换为整型。
    b.浮点数也会被转换为整型。意味着其小数部分会被舍去。
    c.布尔类型也会被转为整型。即 键名为 true 转为 1，false 转为 0.
    d.Null会被转换为空串，即键名null 实际会被存储为 “”.
    e.数组和对象不能作为键名。坚持这么做会导致警告 ：Illegal offset type 。

  2. 清除 array 里的元素

正如我们所见使用unset 并没有重置array的 索引。

简单介绍一下常用的对数组操作函数

  1. in_array()和 array_key_exists() 判断array中是否存在某个value和key。

  2. 数组的合并函数 array_merge()和 array_combine()

  5. 把数组转换成变量 extract()函数

  6. array_search()在数组中查找值是否存在,同in_array()类似

    通过上面的对比我们可以看出，array_search()查找到某值的时候会回返对应的key，而in_array()
    会返回true

  7. 获取array中的所有key值 和 value值

字符串的操作

对字符串的操作无论是在php中还是在lua中大概分为两个方面
1,对字符串的基本操作。如：字符串的长度，大小写的互换，字符串分割等等
2,字符串的正则表达式。包括 字符串的匹配查找，和字符串的替换。

  1. 字符串的基本操作
    a) substr() 按pos切割字符串
    b) wordwrap() 让字符串强制换行在某个位置上
    c) strtolower() 所有字符转换成小写
    d) strtoupper() 所有字符转换成大写
    e) ucwords() 首字母大写
    f) strlen() 字符串长度
    g) substr_count() 某字符串在字符串中出现的次数
    h) str_word_count() 统计某个单词出现的次数
    i) strstr() 某字符串在字符串中出现的位置
    g) strpos() 与 strstr()相同 ，建议 strpos
    k) str_repeat() 重复字符串
    l) substr_replace() 字符串替换
    m) strcmp() strcasecmp() 字符串比较

  2. 字符串的正则表达式
    a)基本模式 ^表示开始,$表示结束
    b)字符集 [a-z] [A-Z] [a-zA-Z] [0-9] [^a-z], 例如：^[^0-9][0-9]$ 是和 “ab2”,”&a”匹配的
    c)php 内置字符集
    [[:alpha:]]任何字母
    [[:digit:]]任何数字
    [[:alnum:]]任何字母和数字
    [[:space:]]任何白字符
    [[:upper:]]任何大写字母
    [[:lower:]]任何小写字母
    [[:punct:]]任何标点符号
    [[:xdigit:]]任何十六进制的数字，相当于[0-9a-fA-F]
    d)一条竖线表示选择分支 例如：想要匹配com，org，cn 可以写成com|org|cn
    e)重复 {num} 表示匹配前面内容num次 ^a{2,4}$表示和aa aaaa匹配

Lua math 库

rounding functions

  1. 向上取整 math.ceil(2.1) --结果为 3
  2. 向下取整 math.floor(2.8) --结果为2

trigonometric functions

  1. 正弦函数 math.sin(math.pi/6) --结果为 0.5
  2. 余弦函数 math.cos(math.pi/3) --结果为 0.5
  3. 正切函数 math.tan(math.pi/4) --结果为 1.0
  4. 反正弦函数 math.asin(0.5) --结果为 math.pi/6
  5. 反余弦函数 math.acons(0.5) --结果为 math.pi/3
  6. 反正切函数 math.atan(1) --结果为 math.pi/4
    注：三角函数的参数都是以弧度为参数，所以我们经常用到radian(弧度)到degress（角度）的转换
  7. 弧度到角度的转化 math.deg(math.pi/2) --结果为 90.0
  8. 角度到弧度的转化 math.rad(60) --结果为 math.pi/3

random functions

  1. 随机函数 math.random() --结果为 interval[0,1) 之间的小数
    注:当然也可以传入两个参数这样也会返回这个区间的随机一个数 例如：math.random(1,100) 会返回1-100的值,也可以传入一个值 n
    这时将返回1 - n的随机值
  2. 设置随机种子 math.randomseed(os.time())

other functions

  1. 获取最大值 math.max(1,2,5,10) --结果为 10
  2. 获取最小值 math.min(1,2,5,10) --结果为 1
  3. 获取绝对值 math.abs(-2015) --结果为2015
  4. 计算x的y次幂 math.pow(x,y)

Patterns

  1. “%d” 匹配任何数字 例如:

  2. “%A” 代表着非字母的字符

  3. “[AEIOUaeiou]” 代表着匹配原音字符

  4. “[^0-4]” 代表着匹配非0-4之间的数字

  5. “^%d+” 代表匹配开头连续的数字

  6. “%s*” 代表匹配0个或多个空格

  7. “[_%w]”代表着匹配 下划线或字符

  8. “.*” 代表匹配最大长度

  9. “.-“ 代表着匹配最短长度

  10. “?” 表示匹配前面的字符0次或1次 “[+-]?%d+” 表示匹配一个正数或负数

  11. “%b()” 表示匹配()之间的东西 也称 blanced string

  12. “%f” 表示匹配边界字符，frontier string
