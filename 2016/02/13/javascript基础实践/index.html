javascript基础实践

  • javascript

概述

cocoscreator的推出javascript为唯一指定脚本，以及高效轻量的Node.js 在后端的普及（最近的项目就有用到）,当然我认为最有魅力的是Node.js的包管理器NPM 号称是全球最大的开源生态系统。下面我主要针对JavaScript 有别于 c++ /c 的地方做一下实践。

变量

  1. 布尔类型的转换。两个否定操作符"!!"可以显示的将数字或字符串转为布尔值

  2. Number 转化成 String 先将16进制或8进制的数转为10进制 然后在转为 字符串

  3. 将字符串转为数字，可以用parseInt和parseFloat函数

相等与相同

  1. 相等==可以类型不同内容相同，相同===要求比较严格，内容和类型都要一样。

对象

javascript的对象包括 有一些与数据类型平行的对象,String、Boolean、Number 还有三种其他功能内建对象 Math,Data,及RegExp(正则表达式)。另外一种内建集合对象Array。

String

  1. 当在基本类型上调用对象的方法时，js将创建基本数据类型的对象实例，执行方法调用，并毁临时对象，该原则同样适用于Boolean 和 Number 。

  2. new 关键字是显示创建对象的关键，如果省略后将是基本类型。创建对象后可用valueOf()方法访问对应的基本类型数值。

  3. String对象的一些方法。

正则表达式

  1. RegExp对象可以显示创建也可以通过文字量的方式创建。

  2. 正则表达式只有两个实例方法 test和exec ，test方法判断传入的参数是否匹配正则表达式，exec方法返回执行结果。

  3. 如果想不区大小写可以在正则表达式后面添加标识符 i，与 i类似的表示符还有g(全局匹配标识符)，m（多行匹配标识符）。

正则表达式的应用

在正则表达式中 *表示匹配0次或多次，+表示匹配1次或多次，?表示匹配0次或1次，.表示字符只出现1次。

  1. RegExp用于字符串替换

  2. 贪婪匹配.*和 延迟匹配 .*?

  3. \D 是非数字，\d 是数字，\s 是空白符。

  4. ^ 脱字符有两个用途一个表示‘非’（非数字也可以表示为[^0-9]），另一个用途是用来表示一行的开始与$(表示一行的结束)相对应。

  5. ()可以用来保留匹配结果。$1-$9这些特殊的符号保存所匹配的字符串

注意：返回结果有两条记录第一条是正则表达式的匹配结果，第二条是括号中的值

  1. 使用()进行所匹配的字符串的交换

注意：\w匹配任何数字或字母 等同于 [A-Za-z0-9] \W 表示非数字或字母

Array

  1. array的创建

注意：与String Number 对象不同的是js会立即将基本类型转换为Array对象变量

  1. array的一些函数

  2. 把array当做队列和栈使用

  3. array数组的遍历

函数

  1. 数组的传参是按引用，字符串是copy

  2. 函数的创建和声明

注意: 使用new创建函数和声明函数不太一样，它将动态创建一个佚名函数，当其被调用后，该函数将被自动删除。
如果在循环语句中使用那么将意味着每次循环将创建一次函数，声明式静态函数只会被创建一次。

结束语
